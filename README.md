## Installation

### RubyGems

```
$ gem install fluent-plugin-flatten-types
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-flatten-types"
```

And then execute:

```
$ bundle
```

You can generate configuration template:

```
$ fluent-plugin-config-format filter flatten_types
```
